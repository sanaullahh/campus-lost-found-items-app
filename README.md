# campus-lost-found-items-app
A campus-wide Lost &amp; Found system that allows students to post lost items, report found items, and chat with item owners to help return belongings.

Campus Lost & Found - Offline Project
This is a single-page offline app that uses localStorage to save items (no server needed).
