Campus Lost & Found - Offline Project
This is a single-page offline app that uses localStorage to save items (no server needed).
How to use:
1. Unzip the folder.
2. Open index.html in your browser.
3. Use 'Add Item' to post lost or found items. Images are stored locally as data URLs.
4. Data persists in your browser's localStorage.

Teacher-friendly: simple, practical, no backend required for demo.
